# LinkOpener

Simple Python-script to open links from the terminal.

## Possibilities

- Open multiple links at once
- Support for HTTPS (default) and HTTP protocols

## Example

```bash
lopen github.com # will be opened github
lopen --http darknet.ru # will be opened http://darknet.ru
lopen chat.deepseek.com gemine.google.com # will be opened deepseek and gemine
```
